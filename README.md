# SO-tp1

### Instrucciones de compilación y ejecución

Para compilar el programa:
```sh
make clean
make
```
Para ejecutar:

1era forma:
```sh
./master <files_path> | ./view
```
2da. forma:
```sh
./master <files_path>
./view <file_quantity>
```
siendo <file_paths> las rutas de los archivos .cnf a procesar y  <file_quantity> la cantidad de archivos que el master procesó
